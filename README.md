# Pawzi — сайт лакомств для собак

Учебный проект: адаптивный многостраничный сайт интернет-магазина товаров для собак.
Написан вручную на HTML, CSS и JavaScript **без использования нейросетей и AI-помощников** — задача ставилась как тренировка ручной верстки и структурирования крупного сайта.

## Статус
Макет полностью свёрстан, финальная доработка (мелкие правки интерактивности и адаптива) не завершена. Проект не запущен и не планируется к публикации — используется как демонстрация навыков вёрстки.

## Стек
- HTML5
- CSS3 (адаптивная вёрстка)
- JavaScript (без фреймворков и сборщиков)

## Реализованные страницы
- Каталог товаров
- Условия доставки и политика доставки
- FAQ
- Информация о компании
- Точки продаж / локации
- Оптовые продажи
- Политика возврата
- Сертификация безопасности
- Соцсети

## Что демонстрирует проект
- Ручная адаптивная вёрстка многостраничного сайта без готовых шаблонов и конструкторов
- Проработка типовых для e-commerce разделов (оплата, доставка, возврат, FAQ)
- Организация структуры проекта и файловой системы
